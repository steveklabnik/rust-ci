Add a .travis.yml file to your project.

Here's an example using the Rust package manager cargo for building. It installs cargo from Corey Richardson's PPA. Using cargo for building is not required. 

install:
  - curl http://www.rust-lang.org/rustup.sh | sudo sh -
script:
  - cargo build --verbose
  - cargo test --verbose
env:
  - LD_LIBRARY_PATH=/usr/local/lib
